# Day3

+ Literals and Variable Declaration
A variable is a literal assigned to an identifier, so you can reference and use it later in the program.

## Declaration types:

* var
* let
* const

## var syntax:
+ var x; //declaration and initialization
x = "Hello JS";

## let syntax:
+ let x;
x = "Hello JS";

+ All in one 
let y = "Hello JS";

## const syntax:
+ const y = "Hello JS";
