# Day 9

Types and Numbers in JS

## Js is divided into two types:
+ primitive types:
 + numbers
 + text or string
 + Boolean
 + null
 + undefined 

+ Object types:
 - global object
 - array
 - function
 - Classes & Constructors
 - garbage collection

## Javascript Numbers

It has only one type of number. Numbers can be written with or without decimals.
